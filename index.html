FarshLang Docs

FarshLang это интерпретируемый функциональный язык программирования, который написан студентами 2-го курса Московского Авиационного Института в рамках лабораторной работы по дисциплине "Логическое и функциональное программирование".

Руководство по синтаксису
Программы на языке FarshLang представляют собой строки. Для определения чего-либо необходимо поставить маркер ! (let в F#), далее пишется индентификатор, который является последовательностью ASCII-символов. Далее мы пишем определение в квадратных скобках. В скобках можем писать любые необходимые выражения. Язык поддерживает аппликацию, которая пишется в фигурных скобках. Также в языке есть поддержка рекурсии. Для написания рекурсивной функции необходимо перед маркером ! написать маркер @. После определения функции ставится маркер ~, а после него выражение, в котором функция применяется. Также в языке есть поддержка условных операторов, с помощью маркера ? можно написать условие которое будет проверяться, далее маркер & - если условие истинно, : - если условие ложно. В языке есть математические операции: сложение, вычитание, умножение, деление, а также строгие и нестрогие операторы сравнения. Помимо стандартных математических операций, есть встроенные математические функции синуса (sin), косинуса (cos) и натурального логарифма (ln). Все арифметические операции представлены в префиксной записи. Функции принимают лишь один аргумент. Так есть поддержка лямбда-функций.

Стэк технологий для реализации
В реализации данного языка использовалась библиотека FParsec, которая выстраивала синтаксическое дерево. Основной язык реализации - F# в среде dotnet.

Применение языка
Язык может применяться для реализации элементарных математических функций с одним аргументом. Код выглядит необычно и компактно, так как вся программа это одна строка.

Клонируем репозиторий
$ git clone git@github.com:MAILabs-Edu-2024/fp-compiler-lab-save_our_souls.git
Переходим в проект
$ cd project_name
Реконструируем
$ dotnet restore
Собираем
$ dotnet build
Запускаем
$ dotnet run

Инкримент (каррирование):
! inc [{+ 1}] ~ {inc 10}
Инкримент (лямбда-функция):
! inc [$ x = [ {{+ x} 1} ] ] ~ {inc 4}
Факториал:
@! factorial [$ x = [? {{< x} 2} : 1 % {{* {factorial {{- x} 1}}} x} ] ] ~ {factorial 5}
Числа Фибоначчи:
@! fib [$x = [? {{> x} 2} & { {+ {fib {{- x} 2}}} {fib {{- x} 1}} } : 1] ] ~ {fib 6}
