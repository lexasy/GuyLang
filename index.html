GuyLang Docs

GuyLang это интерпретируемый функциональный язык программирования, который написан студентами 2-го курса Московского Авиационного Института.

Язык не имеет каких-то продвинутых возможностей. Написан с использованием библиотеки FParsec.

$ dotnet addpackage какая-то хуета

Факториал:
@! factorial [$ x = [? {{< x} 2} : 1 % {{* {factorial {{- x} 1}}} x} ] ] ~ {factorial 5}
